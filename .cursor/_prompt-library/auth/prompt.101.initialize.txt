아래의 조건을 모두 적용하여, 아래의 요구사항을 모두 구현할 것.
구현 결과를 체크리스트로 반환할 것.

==============================================

조건-커서룰) 아래의 커서룰을 적용하여 작업하고, 이 작업이 끝나면 해당 rules 적용 결과를 체크리스트로 반환할 것.
            - .cursor/rules/01-common.mdc

==============================================

조건-파일경로) 구현될 TS 파일경로: @dobby-review/utils/src/supabase/initialize/index.ts

==============================================

핵심요구사항) supabase 클라이언트 초기화 유틸을 구현할 것.
            1. 기본 클라이언트
                - 환경변수: SUPABASE_URL, SUPABASE_ANON_KEY
                - 인증옵션: 세션 자동 저장/자동 갱신 비활성화
            2. 토큰 기반 클라이언트
                - 입력값: access token 필수
                - 헤더형식: Authorization Bearer
                - 인증옵션: 기본 클라이언트와 동일 정책
            3. 관리자 클라이언트
                - 환경변수: SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY
                - 누락처리: 관리자 키 누락 시 즉시 실패
                - 사용범위: 서버 전용
            4. 노출 정책
                - 기본 클라이언트: 생성 함수 + 재사용 인스턴스(싱글톤) 노출
                - 관리자 클라이언트: 생성 함수 + 재사용 인스턴스(싱글톤) 노출
                - 토큰 기반 클라이언트: 생성 함수만 노출, 인스턴스(싱글톤) 노출 금지
