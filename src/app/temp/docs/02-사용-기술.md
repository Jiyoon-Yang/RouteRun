# 사용 기술

코스 등록 테스트(/temp)에서 사용한 기술·도구·개념을 정리했습니다.

---

## 1. 프레임워크·런타임

| 기술 | 용도 |
|------|------|
| **Next.js 14 (App Router)** | 라우팅(`/temp`), 페이지·컴포넌트 구조 |
| **React 18** | 컴포넌트, `useState` / `useRef` / `useEffect` 로 상태·부수 효과 처리 |
| **TypeScript** | 타입 정의, `kakao.d.ts` 로 전역 `window.kakao` 타입 보강 |

---

## 2. 카카오맵 (Kakao Maps JavaScript API)

| 항목 | 설명 |
|------|------|
| **로드 방식** | CDN 스크립트 (`https://dapi.kakao.com/v2/maps/sdk.js?appkey=...&autoload=false`) |
| **Next.js 연동** | `next/script` 의 `Script` 컴포넌트로 로드, `strategy="afterInteractive"` |
| **초기화** | 스크립트 로드 후 `kakao.maps.load(callback)` 안에서 `new kakao.maps.Map(container, options)` 로 지도 생성 |
| **사용 API** | `Map`, `LatLng`, `Polyline`, `Marker`, `event.addListener` (click, mouseover, mouseout) |

**키 종류**: 반드시 **JavaScript 키** 사용. (REST API 키 사용 시 "wrong appKey format" 오류 발생.)

**필수 설정 (카카오 개발자 콘솔)**  
- 앱에서 **지도/로컬(OPEN_MAP_AND_LOCAL)** 서비스 사용 설정  
- **플랫폼 → Web** 등록, **사이트 도메인**에 `http://localhost:3000` 등 추가  

---

## 3. Next.js 관련

| 항목 | 설명 |
|------|------|
| **Client Component** | 지도·이벤트·브라우저 API 사용을 위해 `MapTest` 상단에 `"use client"` 지정 |
| **환경 변수** | 브라우저에서 읽으려면 `NEXT_PUBLIC_` 접두사 필요. `NEXT_PUBLIC_KAKAO_MAP_API_KEY` 사용 |
| **Script** | `next/script` 로 카카오맵 SDK 삽입, `onLoad` 에서 로드 완료 후 지도 초기화 트리거 |

---

## 4. 스타일

| 항목 | 설명 |
|------|------|
| **Tailwind CSS** | 레이아웃·간격·색상·테두리 (블랙·화이트·회색 계열로 단순 구성) |
| **디자인 원칙** | 심플, 블랙 앤 화이트. 지도 영역 `h-[60vh]`, 하단 좌표 목록 등 |

---

## 5. 상태·참조 관리

| 도구 | 용도 |
|------|------|
| **useState** | `points`(클릭 좌표 배열), `scriptLoaded`, `mapReady`, `courseId` |
| **useRef** | `mapContainerRef`(지도 div), `mapRef`(지도 인스턴스), `polylineRef`, `markersRef`, `pointsRef`(클릭 핸들러에서 최신 points 참조) |

`pointsRef` 를 두는 이유: 지도 `click` 리스너는 한 번 등록되면 클로저로 예전 `points` 를 보게 될 수 있어, 항상 최신 배열을 참조하도록 하기 위함.

---

## 6. Supabase (현재 사용 범위)

- **역할**: 저장할 **테이블 구조에 맞춘 데이터 형태**만 준비.
- **실제 insert**: 이 테스트에서는 하지 않음. `toSupabaseCoursePoints()` 반환값을 콘솔·화면에서 확인하는 수준.
- **테이블 컬럼**: id(uuid, 자동), course_id, point_type(START/WAYPOINT/END), latitude, longitude, sequence.

---

## 7. 타입 정의 (kakao.d.ts)

- **위치**: `src/types/kakao.d.ts`
- **목적**: `window.kakao` 및 `kakao.maps.Map`, `Polyline`, `Marker`, `LatLng`, `event.addListener` 등 사용하는 API에 대한 TypeScript 타입 선언.
- **특징**: SDK는 npm이 아니라 스크립트로 로드되므로, 프로젝트에서 직접 선언해 두고 사용.

위 기술들이 [03~05번 문서](./03-기능-지도와-클릭.md)의 기능별 구현에 어떻게 쓰였는지 연결됩니다.
