## 한 줄 요약

**지도에서 출발지 → 경유지 → 도착지를 순서대로 클릭하면, 그 좌표가 선으로 연결되고 Supabase에 넣을 수 있는 형태로 변환되는 테스트 페이지**입니다.

---

## 사용 방법

1. 개발 서버 실행: `npm run dev`
2. 브라우저에서 **http://localhost:3000/temp** 접속
3. 지도를 클릭해 **출발지 → 경유지(여러 개 가능) → 도착지** 순서로 좌표 지정
4. 도착지를 출발지와 같은 위치로 하려면: **출발지 마커에 마우스를 올린 뒤**(마커가 진해짐) **클릭**
5. 개발자 도구 콘솔에서 클릭 좌표와 Supabase 저장용 데이터 확인 가능

---

## 필요한 설정

- **환경 변수**: `.env`에 `NEXT_PUBLIC_KAKAO_MAP_API_KEY`(카카오맵 **JavaScript 키**) 설정
- **카카오 개발자 콘솔**: 앱에서 **지도/로컬** 서비스 사용 설정, **Web 플랫폼**·사이트 도메인 등록

자세한 내용은 [02-사용-기술.md](./02-사용-기술.md) 참고.
