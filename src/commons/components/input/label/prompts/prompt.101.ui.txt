핵심요구사항)
- `label` 요소 기반으로 구현하고 `htmlFor`를 props로 받아 연결 가능하게 할 것.
- `required` 옵션 지원(`*` 표시 또는 스타일 토큰 반영).
- 상태는 최소 `default | error | disabled`를 지원할 것.
- 라벨 타이포는 단일 스케일로 고정한다(별도 size variant 없음).
- Input 조합 컴포넌트에서 재사용 가능한 단일 책임 컴포넌트로 구현할 것.
