핵심요구사항)
- 네이티브 `<input>` 기반으로 구현할 것.
- placeholder 텍스트 스타일은 상태별로 일관되게 적용할 것.
- 상태는 `default | hover | filled | active | error | disabled`를 지원할 것.
- 좌/우 아이콘은 `showLeftIcon` / `showRightIcon` 불린으로 노출 여부를 제어할 것(기본 true).
- 필드 높이/패딩은 단일 스케일로 고정한다(별도 size variant 없음).
- `disabled` 시 상호작용 차단 및 시각 상태를 명확히 표현할 것.
