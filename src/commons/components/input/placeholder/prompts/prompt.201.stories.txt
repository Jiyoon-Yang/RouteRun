구현 후 결과를 체크리스트로 반환할 것.

==============================================

조건-커서룰)
- 아래 커서룰을 적용하고, 작업 종료 시 적용 결과를 체크리스트로 반환할 것.
  - @01-common.mdc

==============================================

조건-파일경로)
- 컴포넌트 TSX: `src/commons/components/input/placeholder/index.tsx`
- 컴포넌트 CSS: `src/commons/components/input/placeholder/styles.module.css`
- 스토리 구현: `src/commons/components/input/placeholder/index.stories.tsx`
- UI 구현 기준 프롬프트: `src/commons/components/input/placeholder/prompts/prompt.101.ui.txt`

- `.storybook/main.ts`의 `addons`에 등록할 것.

==============================================

조건-아이콘)

- `lucide-react`의 `Scan` 컴포넌트를 import하여 `leftIcon` / `rightIcon`에 전달할 것 (컴포넌트는 `FieldLucideIcon`으로 래핑됨).
