핵심요구사항) 구글 로그인 트리거 훅을 구현할 것.
            1. 트리거 역할
                - 대상: 구글 로그인 시작 API 호출
            2. 성공 처리
                - 조건: 응답 성공 + OAuth URL 존재
                - 동작: 해당 URL로 즉시 이동
            3. 파라미터 전달
                - 항목: returnTo
                - 목적: 로그인 완료 후 복귀 경로 유지
            4. 상태 노출
                - 항목: 요청 중 상태 + 실패 상태
                - 목적: 화면에서 로딩/오류 처리 가능

추가-요구사항)
            - 호출 경로: src/app/api/auth/Oath-login.ts API만 사용
            - 금지 사항: Supabase SDK 직접 호출
